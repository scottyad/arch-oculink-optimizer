# Arch Linux Oculink eGPU Optimizer

Automated bare-metal configuration manifests and performance tracking tools to stabilize an external NVIDIA dGPU connected via an Oculink Gen4 link under the `linux-zen` kernel.

## Installation & Deployment

```bash
git clone [https://github.com/scottyad/arch-oculink-optimizer](https://github.com/scottyad/arch-oculink-optimizer)
cd arch-oculink-optimizer
sudo ./optimize-system.sh
